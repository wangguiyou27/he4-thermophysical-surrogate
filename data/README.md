# Data note

The full helium-4 datasets are not included in this repository.

They were generated from NIST REFPROP over the single-phase domain:

- Temperature: 2.2-300 K
- Pressure: 0.001-3 MPa
- Fluid: helium-4

REFPROP is licensed software and is not redistributed here. To reproduce the dataset, install REFPROP locally and run the data-generation scripts in `scripts/`.

Only final summary tables and figures are included under `paper_outputs/he4_2p2_hybrid_paper/`.
