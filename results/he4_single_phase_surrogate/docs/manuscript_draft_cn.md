# 基于临界特征神经网络与局部混合修正的单相氦-4热物性快速预测模型

## 摘要

氦-4热物性在低温制冷、超导设备冷却、低温换热器和氦循环系统仿真中具有重要作用。REFPROP 等权威物性程序能够提供高精度物性数据，但在大规模参数扫描、优化设计和瞬态迭代计算中，频繁调用物性函数会带来明显的计算开销。为提高单相氦-4物性计算效率，并控制工程应用中可能导致迭代失稳的极端误差，本文基于 REFPROP 构建了覆盖 2.2-300 K、0.001-3 MPa 的单相氦-4热物性数据集，建立了包含相区标识和临界特征的多输出神经网络模型，并进一步提出局部混合修正方法以抑制尾部误差。模型以温度、压力、相区标识、约化温度、约化压力和临界距离等变量作为输入，同时预测密度、熵、焓、声速、定容比热、定压比热、黏度和导热系数八个物性。结果表明，最终混合模型的整体平均绝对百分比误差为 0.1307%，P99 误差为 0.9414%，最大相对误差为 4.6514%，所有测试点误差均小于 5%。在 100000 个状态点的批量测试中，混合模型的计算速度约为 REFPROP 直接调用的 17.45 倍，而相对于纯神经网络模型仅增加约 4.50% 的时间开销。与 Random Forest、Extra Trees、XGBoost 和 Gaussian RBF Ridge 等基线模型相比，所提出模型在计算速度、尾部误差和工程可靠性之间取得了更优平衡，可为低温工程仿真中的快速物性查询提供有效工具。

**关键词：** 氦-4；热物性；神经网络；REFPROP；低温工程；混合模型；临界特征

## 1. Introduction

氦-4是低温工程中最重要的工质之一，广泛应用于超导磁体冷却、低温制冷机、低温换热器、氦循环系统以及低温实验装置中。由于氦在低温区域具有明显的非理想性，其密度、比热、声速和输运性质会随温度和压力发生剧烈变化，尤其在低温液态区、近临界区以及相界附近，物性的非线性特征更加突出。因此，准确计算氦-4热物性是低温系统设计、性能预测和数值仿真的基础。

目前，REFPROP 等物性程序通常被视为工程计算中的高可信数据来源。REFPROP 基于高精度状态方程和输运性质关联式，可以在较宽温压范围内提供可靠的热物性数据。然而，在低温设备的动态仿真、参数优化、控制策略计算和大规模设计空间搜索中，物性函数往往需要被反复调用。若每一步迭代均直接调用 REFPROP，计算效率可能成为限制因素。对于需要大量状态点查询的工程任务，建立快速、稳定且足够准确的物性代理模型具有现实意义。

近年来，机器学习方法已被用于流体热物性预测。相比传统经验关联式，神经网络模型能够从高维数据中学习复杂非线性映射，并在训练后实现快速批量推理。然而，低温氦物性建模仍存在若干困难。首先，低温和近临界区域的物性变化梯度大，普通回归模型容易在局部区域产生较大误差。其次，多物性输出之间量纲和数值尺度差异显著，若不进行适当变换和加权训练，部分物性可能被模型忽略。第三，工程计算不仅关注平均误差，也高度关注最大误差和尾部误差，因为单个异常物性点可能导致迭代计算不收敛或传播错误状态。

针对上述问题，本文建立了面向单相氦-4物性快速计算的神经网络与局部混合修正模型。本文主要贡献如下：

1. 基于 REFPROP 生成并筛选了覆盖 2.2-300 K、0.001-3 MPa 的单相氦-4数据集，包含液相、气相和超临界等单相状态。
2. 构建了包含相区标识和临界特征的多输出神经网络模型，使模型能够更好地处理低温区和临界附近的强非线性。
3. 提出局部混合修正方法，对纯神经网络模型的高风险区域进行局部补强，将所有物性的最大相对误差控制在 5% 以下。
4. 从平均误差、P99 误差、最大误差、R2、阈值内比例和计算速度等多个角度，对最终模型、传统机器学习模型和 REFPROP 直接调用进行对比。

## 2. Dataset Generation and Phase Filtering

本文数据由 REFPROP 生成，工质为氦-4。数据覆盖温度 2.2-300 K、压力 0.001-3 MPa。输出物性包括密度、熵、焓、声速、定容比热、定压比热、黏度和导热系数。相比早期 5.5 K 以上温区的模型，本文将温度下限扩展至 2.2 K，使模型覆盖更低温的氦-4单相区域，从而更适合低温工程应用。

由于本文目标是建立单相物性代理模型，因此数据生成后需要进行相区筛选。对于亚临界区域，温度和压力可能对应液相、气相或两相状态。两相区物性通常不仅由温度和压力唯一确定，还与干度、相分布或混合规则有关。若将两相数据直接并入以温度和压力为输入的单值回归模型，会引入物理歧义。因此，本文根据 REFPROP 相态信息和饱和压力关系去除两相区及近饱和风险点，仅保留单相状态。最终数据包括 subcooled liquid、superheated vapor、single-phase gas 和 supercritical 等区域。

## 3. Methodology

### 3.1 Multi-output ANN surrogate

本文采用多输出多层感知机作为基础物性代理模型。模型一次输入一个状态点，同时输出八个物性。相比为每个物性单独训练模型，多输出模型可以共享温度、压力和相区信息中的热力学特征，有助于减少模型数量并提高部署便利性。

若仅以温度和压力作为输入，模型需要自行从数据中学习相区差异和临界附近的复杂结构。对于氦-4而言，临界区附近的物性变化尤其剧烈，纯 T-P 输入模型容易在局部区域出现较大尾部误差。为降低建模难度，本文在温度和压力基础上引入相区标识和临界特征。最终模型输入包括：

- Temperature
- Pressure
- phase_code
- reduced_temperature
- reduced_pressure
- critical_distance
- inverse_critical_distance
- abs_reduced_temperature_offset
- abs_reduced_pressure_offset

其中，约化温度和约化压力用于刻画状态点相对于临界点的位置，临界距离及其倒数用于增强模型对临界附近强非线性区域的感知能力。相区标识则向模型提供单相状态类别信息，有助于减少液相、气相和超临界状态之间的混淆。

### 3.2 Target transformation and weighted training

八个目标物性的量纲和数量级差异较大。例如，密度、焓、黏度和导热系数的数值范围差异明显，熵和焓在低温区域还可能接近零或发生符号变化。为改善训练稳定性，本文对部分正值物性采用对数变换，对存在参考零点附近变化的目标采用适当的平移对数变换，并在模型输出后通过反变换恢复物理量。

训练过程中对输入和输出均进行标准化处理。损失函数采用加权均方误差形式，对低温区、临界附近区域以及历史误差较大的物性给予更高权重。该策略的目的不是单纯降低全局平均误差，而是提升模型在复杂区域的可靠性，减少可能影响工程迭代计算的异常点。

### 3.3 Local hybrid correction

纯神经网络模型具有较高的平均精度和很快的推理速度，但在低温、临界区和物性接近零值的局部区域仍可能产生较大的相对误差。对于工程仿真而言，少量极端误差点可能比平均误差更重要。因此，本文在临界特征 ANN 基础上引入局部混合修正方法。

局部混合修正的基本思想是：以 ANN 作为全域主模型，对于已知高风险区域，预先构建局部高密度修正网格，并在推理时对这些区域进行轻量插值修正。该方法不需要在每个推理点实时调用 REFPROP，因此不同于传统的逐点 REFPROP fallback。局部修正只作用于低温液体、近临界低压区、临界 Cp 峰、低密度气体、低温近零焓/熵等高风险区域，从而在控制尾部误差的同时尽量保持神经网络的速度优势。

## 4. Accuracy Evaluation

### 4.1 Overall accuracy

Table 2 给出了临界特征 ANN 与局部 hybrid 模型的整体指标对比。纯 ANN 模型已经具有较高平均精度，其 MAPE 为 0.1655%，P99 误差为 1.4251%。然而，纯 ANN 的最大相对误差达到 46.1014%，说明其在少数局部点上仍存在较大尾部误差。

引入局部 hybrid 修正后，模型整体 MAPE 降低至 0.1307%，P99 误差降低至 0.9414%，最大相对误差降低至 4.6514%。同时，最终 hybrid 模型的 mean R2 达到 0.9999845，within 5% 达到 100%。这表明局部混合修正不仅改善了平均精度，更重要的是显著压制了极端误差。

八个物性的单独评价结果见 Table 3 和 Fig. 3。最终 hybrid 模型在所有物性上均取得较高精度。其中，Cv、Cp、声速和导热系数的平均误差较低，密度、熵和焓虽然在低温和近零参考值区域更难预测，但最大误差仍被控制在 5% 以下。

## 5. Comparison and Computational Efficiency

### 5.1 Comparison with machine-learning baselines

为评估所提出模型的综合性能，本文将当前模型与多种传统机器学习方法进行对比，包括 Random Forest、Extra Trees、XGBoost 和 Gaussian RBF Ridge。基线模型结果见 Table 4 和 Fig. 6。

结果表明，部分树模型和核方法可以取得较低的平均 MAPE。例如 Extra Trees 的 MAPE 为 0.0281%，Gaussian RBF Ridge 的 MAPE 为 0.0480%，Random Forest 的 MAPE 为 0.0538%。然而，这些模型的最大误差明显较大，分别达到 14.4329%、43.9026% 和 26.5785%。此外，这些方法的推理速度也低于当前 MLP 模型。XGBoost 在当前设置下表现较差，MAPE 为 1.4261%，最大误差达到 119.5872%。

这些结果表明，平均误差最低并不意味着模型最适合工程仿真。对于低温系统迭代计算而言，最大误差、P99 误差和速度同样重要。树模型虽然在局部平均拟合上具有优势，但其预测连续性、尾部可靠性和批量推理速度均不如最终 hybrid 模型。最终 hybrid 模型虽然不是平均 MAPE 最低的模型，但其最大误差小于 5%，并保持较高推理速度，因此在工程可靠性方面更具优势。

### 5.2 Comparison with direct REFPROP calls

除模型精度外，计算效率也是本文关注的核心指标。Table 5 和 Fig. 7 给出了 ANN、hybrid 和 REFPROP 直接调用的速度对比。在 100000 个状态点的批量测试中，纯 ANN 耗时 0.403249 s，吞吐率为 247985.7 points/s；hybrid 模型耗时 0.421392 s，吞吐率为 237308.6 points/s；REFPROP 直接调用耗时 7.352703 s，吞吐率为 13600.4 points/s。

相对于 REFPROP，纯 ANN 的加速比为 18.23 倍，hybrid 模型的加速比为 17.45 倍。与纯 ANN 相比，hybrid 模型仅增加约 4.50% 的计算时间，却将最大误差从 46.1014% 降至 4.6514%。因此，局部 hybrid 修正并未明显削弱神经网络模型的速度优势，而是以较小时间代价显著提升了尾部可靠性。

本文还对预生成数据库插值方法进行了初步比较。结果显示，基于 KDTree 的预计算数据插值在速度上可以超过 ANN，但其最大误差明显较大，部分设置下最大误差超过 100%。这说明查表插值虽然是一种快速路线，但在低温和近临界强非线性区域难以同时保证全域尾部精度。相比之下，本文 hybrid 模型在保持 REFPROP 约 17.45 倍加速的同时，将所有物性的最大误差控制在 5% 以下，更适合作为工程仿真的可靠物性代理。

## 6. Discussion

### 6.1 关于温度下限选择

早期模型采用 5.5 K 作为温度下限，主要是为了避开低温复杂区域并快速验证方法可行性。然而，5.5 K 对低温工程而言具有一定人为性，也难以覆盖更低温氦系统中的关键状态。本文将温度下限扩展至 2.2 K，使模型更加贴近低温工程应用需求。低温扩展显著增加了建模难度，但通过相区标识、临界特征和局部 hybrid 修正，最终模型仍实现了所有物性最大误差小于 5%。

需要说明的是，本文并不声称建立超流氦专门模型，也不处理两相混合区。本文关注的是 REFPROP 数据范围内的单相氦-4状态，并通过相区筛选避免两相物性的物理歧义。

### 6.2 关于两相区

两相区物性通常不能仅由温度和压力唯一确定，往往还需要干度或其他相平衡变量。如果直接把两相区加入当前 T-P 单值回归模型，可能导致同一温压状态对应不同混合物性，从而破坏监督学习目标的唯一性。因此，本文选择先构建单相高可靠模型。未来若进一步扩展到两相区，应引入 T-P-x 或其他相态变量，并结合相平衡模型单独处理。

### 6.3 关于输入变量形式

REFPROP 支持多种输入组合，例如 T-P、P-h、P-s、T-rho 等。本文选择 T-P 作为输入，是因为温度和压力是工程仿真中最常见、最直接的状态变量组合之一，也是低温换热器、管路流动和制冷系统计算中高频使用的物性查询形式。本文目标不是替代 REFPROP 的全部输入组合，而是针对高频 T-P 查询构建快速、可靠的代理模型。

对于 P-h、P-s 等输入组合，未来可以通过两种方式扩展：一是分别训练对应输入组合的代理模型；二是利用当前 T-P 模型结合快速反演算法求解温度或压力。后者可以在保持当前模型结构的基础上扩展适用场景，但需要进一步研究反演稳定性和收敛性。

### 6.4 关于工程可靠性

在许多工程仿真中，物性模型的最大误差比平均误差更关键。若某一状态点的比热、密度或焓出现明显异常，可能导致能量方程、动量方程或迭代求解过程发生不稳定。因此，本文不仅报告 MAPE 和 R2，也重点报告 P95、P99、最大误差和 within 5% 指标。最终 hybrid 模型所有物性最大误差均低于 5%，说明其在尾部可靠性方面优于纯 ANN 和其他基线模型。

### 6.5 关于应用算例

本文没有将 GM 脉管制冷机或稀释制冷机算例作为正文核心。原因是这些系统级算例本身需要完整的流动、换热和边界条件验证，容易将论文重点从物性模型转移到制冷机数值模型。当前研究的最强证据来自 REFPROP 基准数据上的全域精度、尾部误差控制和速度提升。因此，系统级算例更适合作为后续工作，在进一步扩展模型温压范围和输入变量形式后开展。

## 7. Conclusions

本文面向低温工程中氦-4物性快速计算需求，建立了覆盖 2.2-300 K、0.001-3 MPa 单相区域的多输出神经网络物性模型，并提出局部 hybrid 修正方法以控制尾部误差。主要结论如下：

1. 基于 REFPROP 构建的单相氦-4数据集覆盖液相、气相和超临界状态，可用于训练温度-压力输入下的多物性代理模型。
2. 引入相区标识和临界特征后，神经网络能够更好地处理低温和近临界区域的强非线性物性变化。
3. 纯 critical-feature ANN 的整体 MAPE 为 0.1655%，但最大误差达到 46.1014%；加入局部 hybrid 修正后，整体 MAPE 降至 0.1307%，P99 误差降至 0.9414%，最大误差降至 4.6514%。
4. 最终 hybrid 模型在密度、熵、焓、声速、Cv、Cp、黏度和导热系数八个物性上的最大误差均小于 5%，within 5% 达到 100%。
5. 在 100000 个状态点批量测试中，hybrid 模型比 REFPROP 直接调用快约 17.45 倍，而相对于纯 ANN 仅慢约 4.50%，体现出较好的速度-可靠性平衡。
6. 与 Random Forest、Extra Trees、XGBoost、Gaussian RBF Ridge 和预生成数据插值方法相比，本文方法在尾部误差控制和工程可靠性方面更具优势。

综上，所提出的临界特征 ANN 与局部 hybrid 修正模型能够在保持较高计算效率的同时显著降低单相氦-4物性预测的极端误差，可作为低温工程仿真、参数扫描和优化计算中的快速物性查询工具。
